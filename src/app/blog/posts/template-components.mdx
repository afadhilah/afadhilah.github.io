## Typography

### Heading (h1 - h6)

Heading otomatis menjadi anchor link yang bisa diklik. Berikut contoh setiap level:

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6

### Paragraph

Ini adalah contoh paragraf biasa. Teks ini akan di-render menggunakan `Text` component dari Once UI dengan variant `body-default-m`. Setiap paragraf memiliki line-height 175% untuk keterbacaan yang optimal.

Paragraf kedua untuk menunjukkan spacing antar paragraf. Kamu bisa menulis **bold**, *italic*, dan ***bold italic*** seperti biasa menggunakan Markdown syntax.

### Custom Heading Component

Selain heading Markdown standar, kamu juga bisa menggunakan `<Heading>` component langsung:

<Heading as="h3" variant="heading-strong-l" marginTop="24" marginBottom="12">
  Custom Heading dengan Variant
</Heading>

<Heading as="h4" variant="display-strong-s" marginBottom="12">
  Display Strong Small
</Heading>

### Text Component

<Text variant="body-default-l" onBackground="neutral-strong" marginBottom="12">
  Text variant: body-default-l — untuk teks yang lebih besar dan menonjol.
</Text>

<Text variant="body-default-s" onBackground="neutral-weak" marginBottom="12">
  Text variant: body-default-s — untuk teks kecil, caption, atau keterangan tambahan.
</Text>

<Text variant="label-strong-m" onBackground="brand-medium" marginBottom="12">
  Text variant: label-strong-m — cocok untuk label atau tag.
</Text>

<Text variant="heading-strong-m" marginBottom="12">
  Text variant: heading-strong-m — heading tanpa anchor link.
</Text>

## Inline Code & Code Block

### Inline Code

Gunakan backtick untuk inline code: `const x = 42;` atau `npm install once-ui`.

Kamu juga bisa pakai component langsung: <InlineCode>InlineCode component</InlineCode>

### Code Block (Markdown Syntax)

Cukup gunakan triple backtick dengan bahasa:

```tsx
import { Button, Column, Row } from "@once-ui-system/core";

export default function MyComponent() {
  return (
    <Column gap="16" padding="24">
      <Row gap="8">
        <Button variant="primary" size="m">
          Primary
        </Button>
        <Button variant="secondary" size="m">
          Secondary
        </Button>
      </Row>
    </Column>
  );
}
```

```css
.container {
  display: flex;
  flex-direction: column;
  gap: 1rem;
  padding: 2rem;
  border-radius: var(--radius-l);
  background: var(--surface-background);
}
```

```python
def fibonacci(n: int) -> list[int]:
    """Generate fibonacci sequence."""
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    return fib[:n]

print(fibonacci(10))
```

### CodeBlock Component (Advanced)

Gunakan `<CodeBlock>` component untuk fitur lebih lengkap seperti highlight baris, label, dan multiple tabs:

<CodeBlock
    marginBottom="16"
    highlight="3-5"
    codes={[
  {
    code:
`// next.config.mjs
const nextConfig = {
  output: "export",
  pageExtensions: ["ts", "tsx", "md", "mdx"],
  images: { unoptimized: true },
};

export default nextConfig;`,
    language: "javascript",
    label: "next.config.mjs"
  }
]} />

#### Multi-tab CodeBlock

<CodeBlock
    marginBottom="16"
    codes={[

## Links

### Internal Link

Navigasi ke halaman lain di portfolio: [Kembali ke Blog](/blog) atau [Halaman About](/about).

### External Link

Link ke situs eksternal: [GitHub](https://github.com) — otomatis terbuka di tab baru.

### Anchor Link

Link ke section di halaman yang sama: [Ke bagian Table](#table).

## Lists

### Unordered List

- Item pertama dalam unordered list
- Item kedua dengan penjelasan lebih panjang yang menunjukkan wrapping
- Item ketiga
- Item keempat

### Ordered List

1. Langkah pertama: Setup project
2. Langkah kedua: Install dependencies
3. Langkah ketiga: Konfigurasi
4. Langkah keempat: Deploy

### Nested List

- Parent item satu
  - Child item A
  - Child item B
- Parent item dua
  - Child item C
  - Child item D
    - Grandchild item
